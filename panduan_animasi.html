Panduan Simulasi

Ini adalah halaman awal untuk animasi dalam web kami.

Keterangan :
  • Tombol Start = Untuk memulai animasi.
  • Tombol Refresh = untuk Refresh animasi dan juga melihat lintasan Parabola sebelum mengklik tombol StartPath.
  • Tombol StartPath = untuk melihat lintasan Parabola.

    • Dalam halaman ini, disediakan isian untuk mengisi sudut Theta dan kecepatan V0
    serta menentukan Motion Speed lalu kita mengisi inputannya.


    Setelah kita mengisi inputannya, kita dapat memulai animasinya dengan tombol "Start"


    Untuk dapat melihat lintasan Parabola, pertama sekali kita mengklik tombol "Refresh"


    Selanjutnya kita klik StartPath, maka akan ditampilkan gerak Parabola seperti
    pada gambar diatas, dengan sudut Theta=45 dan kecepatan V0=80.

    Untuk menciptakan lintasan parabola yang berbeda, kita dapat mengubah sudut Theta
    dan kecepatan V0 serta menentukan Motion Speed seperti pada gambar diatas.

    Lalu, kita mengklik tobol "StartPath" untuk menciptakan lintasan yang berbeda. Dengan tombol "StartPath"
    kita dapat menciptakan banyak lintasan parabola yang berbeda pula.

    Demikian Panduan Animasi dalam web kami. Semoga membantu.
